# horror-game-test

Only group member is Conner Nist

I never managed to come up with a name I was satisfied with so the game is just referred to as "horror game test"
You want to escape a maze that a robot is chasing you throughout, if it reaches you it will stab you and kill you.
It wanders around until it gets near the player which then it begins sprinting at them.

Everything here is wholly original except the code, it was taken from a multitude of sources then modified to
work with what I was trying to do.
